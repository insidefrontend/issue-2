# 現場の ES201x とアーキテクチャの変遷と未来

## 概要 (100文字程度)

フロントエンド激動の時代を経て、キャッチアップに苦しむ現場や、経年劣化に苦しむプロダクトなどが散見されるようになりました。
ここ10年のアーキテクチャの変遷を振り返り、次の世代に何を見定めるべきか明確にします。
